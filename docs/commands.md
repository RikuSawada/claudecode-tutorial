# コマンドリファレンス

Claude Code で使えるコマンド・ショートカットの一覧です。

---

## CLI コマンド（起動時）

ターミナルで `claude` を実行する際に使うコマンドです。

| コマンド | 説明 | 例 |
|--------|------|-----|
| `claude` | 対話モードで起動 | `claude` |
| `claude "プロンプト"` | 初期プロンプト付きで起動 | `claude "このプロジェクトを説明して"` |
| `claude -p "プロンプト"` | 1回だけ実行して終了（非対話） | `claude -p "バグを修正して"` |
| `claude -c` | 直前のセッションを再開 | `claude -c` |
| `claude -r "セッション名"` | 名前またはIDでセッションを再開 | `claude -r "auth-refactor"` |
| `claude update` | 最新バージョンに更新 | `claude update` |
| `claude auth login` | Anthropic アカウントにログイン | `claude auth login` |
| `claude auth logout` | ログアウト | `claude auth logout` |
| `claude auth status` | 認証状態を確認 | `claude auth status` |
| `claude mcp` | MCP サーバーを設定・管理 | `claude mcp` |

---

## CLI フラグ（よく使うもの）

`claude` コマンドに付けるオプションです。

| フラグ | 説明 | 例 |
|------|------|-----|
| `--version`, `-v` | バージョンを表示 | `claude -v` |
| `--model` | 使用するモデルを指定 | `claude --model claude-opus-4-6` |
| `--dangerously-skip-permissions` | すべての承認確認をスキップ（注意して使用） | `claude --dangerously-skip-permissions` |
| `--permission-mode plan` | プランモードで起動（実行せず計画だけ立てる） | `claude --permission-mode plan` |
| `--read-only` | 読み取り専用モードで起動（ファイル変更不可） | `claude --read-only` |
| `--add-dir` | アクセスを許可する追加ディレクトリを指定 | `claude --add-dir ../libs` |
| `--verbose` | 詳細ログを出力 | `claude --verbose` |
| `--worktree`, `-w` | Git worktree に分離して起動 | `claude -w feature-auth` |

## スラッシュコマンド（セッション内）

セッション中に `/コマンド` で呼び出せる命令です。

### よく使うコマンド

| コマンド | 説明 |
|--------|------|
| `/help` | ヘルプを表示 |
| `/clear` | 会話履歴をリセット |
| `/compact` | 長くなった会話を要約して圧縮 |
| `/exit` | セッションを終了 |

### 開発支援

| コマンド | 説明 |
|--------|------|
| `/commit` | 変更内容を解析して git commit を実行 |
| `/review-pr [PR番号]` | Pull Request をレビュー |
| `/init` | `CLAUDE.md` を生成してプロジェクトを初期化 |
| `/plan` | プランモードに切り替え（実装前に計画を確認） |

### 情報・設定

| コマンド | 説明 |
|--------|------|
| `/config` | 設定画面を開く |
| `/status` | バージョン・モデル・接続状態を確認 |
| `/model` | 使用するモデルを変更 |
| `/permissions` | 権限設定を表示・変更 |
| `/memory` | `CLAUDE.md` のメモリファイルを編集 |
| `/mcp` | MCP サーバーの接続状態を確認・管理 |
| `/cost` | 現在のセッションのトークン使用量を確認 |
| `/stats` | 日次使用状況・セッション履歴を表示 |
| `/theme` | カラーテーマを変更 |

### セッション管理

| コマンド | 説明 |
|--------|------|
| `/resume [セッション名]` | 過去のセッションを再開 |
| `/rename [名前]` | 現在のセッションに名前をつける |
| `/rewind` | 会話・コードを前の時点に巻き戻す |
| `/export [ファイル名]` | 会話内容をファイルにエクスポート |
| `/copy` | 最後の回答をクリップボードにコピー |
| `/todos` | 現在の TODO リストを表示 |
| `/tasks` | バックグラウンドタスクを一覧・管理 |

### 診断

| コマンド | 説明 |
|--------|------|
| `/doctor` | インストール状態を診断 |
| `/debug` | セッションのデバッグログを確認 |
| `/context` | 現在のコンテキスト使用状況を視覚化 |

## キーボードショートカット

### 基本操作

| ショートカット | 説明 |
|------------|------|
| `Enter` | メッセージを送信 |
| `Shift+Enter` | 改行（送信しない）※ターミナルによる |
| `↑` / `↓` | 入力履歴を移動 |
| `Ctrl+R` | 入力履歴を逆順検索 |
| `Ctrl+C` | 現在の生成・入力をキャンセル |
| `Ctrl+D` | セッションを終了（EOF） |
| `Ctrl+L` | 画面をクリア（会話履歴は保持） |

### 便利な操作

| ショートカット | 説明 |
|------------|------|
| `Ctrl+O` | ツール実行の詳細表示を切り替え |
| `Ctrl+B` | コマンドをバックグラウンドで実行 |
| `Ctrl+T` | タスクリストの表示を切り替え |
| `Shift+Tab` | 権限モードを切り替え（通常 → Auto-Accept → Plan） |
| `Ctrl+G` | デフォルトのテキストエディタでプロンプトを編集 |
| `Ctrl+V` | クリップボードから画像を貼り付け |
| `Esc` + `Esc` | 会話を前の時点に巻き戻す |

### テキスト編集

| ショートカット | 説明 |
|------------|------|
| `Ctrl+K` | カーソルから行末まで削除 |
| `Ctrl+U` | 行全体を削除 |
| `Ctrl+Y` | 削除したテキストを貼り付け |
| `Alt+B` | カーソルを1単語分戻す |
| `Alt+F` | カーソルを1単語分進める |

## 入力プレフィックス

メッセージの先頭に付ける特殊な文字です。

| プレフィックス | 説明 | 例 |
|------------|------|-----|
| `/` | スラッシュコマンドを呼び出す | `/clear` |
| `!` | bash コマンドを直接実行して結果を会話に追加 | `! git status` |
| `@` | ファイルパスのオートコンプリートを起動 | `@src/auth.ts` |

## Vim モード

`/vim` コマンドまたは `/config` で有効にできます。

### モード切り替え

| キー | 説明 |
|-----|------|
| `i` | INSERT モードに入る |
| `Esc` | NORMAL モードに戻る |

### NORMAL モードでよく使う操作

| キー | 説明 |
|-----|------|
| `h` / `j` / `k` / `l` | 左 / 下 / 上 / 右に移動 |
| `w` / `b` | 次の単語 / 前の単語 |
| `0` / `$` | 行頭 / 行末 |
| `dd` | 行を削除 |
| `yy` | 行をコピー |
| `p` | 貼り付け |
| `u` | 元に戻す |
| `.` | 直前の操作を繰り返す |
