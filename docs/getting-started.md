# Getting Started - セットアップガイド

---

## 必要な環境

- macOS / Linux / Windows (WSL)
- Node.js 18 以上
- npm

---

## インストール

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

---

## 起動方法

```bash
# プロジェクトのディレクトリに移動
cd your-project

# Claude Code を起動
claude
```

---

## 初回セットアップ

1. 起動後、Anthropic アカウントでのログインが求められます
2. ブラウザが開くので、アカウントでログイン
3. 認証完了後、ターミナルに戻ると使用可能になります
